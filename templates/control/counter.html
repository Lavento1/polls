템플릿 태그 - forloop.counter 예제

    {% for item in items %}
  • {{ forloop.counter0 }} -- {{ item }}
    • {% endfor %}
    {% for item in items %} {% if forloop.counter < 3 %}
  • {{ forloop.counter }}번째 줄입니다.
    • {% endif %} {% endfor %}

내장 템플릿 필터(filter) 태그 사용하기

forloop.counter|add:5

    {% for item in items %} {% if forloop.counter < 3 %}
  • {{ forloop.counter|add:5 }}번째 줄입니다.
    • {% endif %} {% endfor %}
